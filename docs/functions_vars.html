Simulador de mémoria cache
Esse projeto foi desenvolvido na disciplina de Introdução a arquitetura e organização de computadores e tem o objetivo de demostrar o comportamento da mémoria cache com diferente técnicas de mapeamento.
 
  • acessos : Cache
  • blocos : Memoria
  • chegada : Linha
  • conteudo : Bloco
  • linhas : Cache
  • numero_do_bloco : Linha
  • refencia_temporal : Linha
  • total_de_usos : Linha
